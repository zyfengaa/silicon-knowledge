# 01 — 数字逻辑基础

> 计算机一切计算的起点。理解 0 和 1 如何变成加法、乘法、浮点数运算。

---

## 本模块内容

| 笔记 | 主题 | 核心问题 |
|------|------|---------|
| 01 | **数制系统** | 二进制、十六进制、补码、IEEE 754 浮点数如何在内存中表示 |
| 02 | **布尔代数** | 与或非运算、布尔定律、卡诺图化简 |
| 03 | **逻辑门** | AND/OR/NOT/NAND/NOR/XOR 门电路、真值表、CMOS 实现简介 |
| 04 | **组合逻辑** | 加法器（半加/全加/超前进位）、多路选择器、译码器、比较器 |
| 05 | **时序逻辑** | SR 锁存器、D 触发器、寄存器组、计数器、时钟与同步 |
| 06 | **算术电路** | 乘法器（Booth 编码、Wallace 树）、除法器 |

## 前置知识

无。只需基本数学基础。

## 建议学习方式

1. 按 `notes/01`, `02`, ... 顺序阅读
2. 读完后运行对应章节的 `code/` 代码，观察输出
3. 动手画真值表和逻辑图
4. 完成 `exercises/` 练习题
5. 选读 `papers/` 中的拓展内容

## 本模块代码

| 文件 | 内容 |
|------|------|
| `python/logic_gates.py` | 用 Python 模拟基本逻辑门的行为 |
| `python/full_adder_sim.py` | 全加器与 4-bit 行波进位加法器 |
| `python/ieee754_demo.py` | IEEE 754 单精度浮点数的位级拆解 |

## 关键产出

- [ ] 能手工将十进制转为二进制/十六进制/补码
- [ ] 会用布尔代数化简简单逻辑表达式
- [ ] 能用基本门搭建半加器和全加器
- [ ] 知道 IEEE 754 浮点数的三个组成部分（符号位/指数/尾数）
- [ ] 理解 D 触发器如何保存 1 bit 状态
